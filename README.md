# 📰 Telegram → Discord News Forwarder

Бот автоматически забирает новые посты из указанных Telegram‑каналов и публикует их в вашем Discord‑канале с красивым оформлением.

## ✨ Возможности

- 🔄 **Автоматическая пересылка** — каждые 15 секунд проверяет каналы на наличие новых постов.
- 📝 **Текст** — отображается в embed-сообщении.
- 🖼️ **Фото** — встраиваются прямо в embed.
- 🎬 **Видео и GIF** — прикрепляются файлами с информацией о длительности.
- 📎 **Документы (файлы)** — прикрепляются.
- 🔗 **Веб‑ссылки** — автоматически выделяются.
- 📲 **Кнопка «Открыть в Telegram»** с прямой ссылкой на оригинальный пост.
- 🎨 **Индивидуальные эмодзи** для каждого канала для быстрого визуального различия.

## 🧰 Требования

- Python 3.9+
- Установленные библиотеки:
  - `discord.py`
  - `python-telegram-bot`
  - `python-dotenv`

Установите зависимости:
```bash
pip install discord.py python-telegram-bot python-dotenv
```

## 🤖 Создание ботов

### Discord бот
1. Перейдите в [Discord Developer Portal](https://discord.com/developers/applications) → New Application.
2. Вкладка **Bot** → Add Bot → скопируйте **Token**.
3. В разделе **Privileged Gateway Intents** включите **MESSAGE CONTENT INTENT** (обязательно!)
4. Пригласите бота на сервер с правами:
   - `Send Messages`
   - `Attach Files`
   - `Embed Links`
5. Включите режим разработчика в Discord (Настройки → Расширенные) и скопируйте **ID канала**, куда бот будет слать новости.

### Telegram бот
1. Напишите [@BotFather](https://t.me/botfather) и создайте нового бота (`/newbot`). Скопируйте **токен**.
2. **Добавьте бота в нужный канал** и **обязательно сделайте администратором** (без этого приватный канал не будет виден).
3. Узнайте ID канала:
   - Перешлите любое сообщение из канала боту [@username_to_id_bot](https://t.me/username_to_id_bot). Он вернёт ID вида `-1001234567890`.

## ⚙️ Настройка

Создайте в корне проекта файл `.env` (без кавычек и пробелов вокруг `=`) и заполните:

```ini
DISCORD_TOKEN=ваш_discord_токен
DISCORD_CHANNEL_ID=*******************
TELEGRAM_BOT_TOKEN=ваш_telegram_токен
TG_CHANNEL_IDS=-*************
CHANNEL_EMOJIS=📰
```

- `TG_CHANNEL_IDS` — ID каналов через запятую **без пробелов**.  
  Пример для нескольких: `-*************,-*************`
- `CHANNEL_EMOJIS` — эмодзи для каждого канала в том же порядке.

## 🚀 Быстрый старт

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/tg2discord.git
cd tg2discord
```
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
   (или вручную `pip install discord.py python-telegram-bot python-dotenv`)
3. Создайте файл `.env` с вашими данными (см. пример выше).
4. Запустите:
```bash
python main.py
```
Бот начнёт опрашивать Telegram каждые 15 секунд и пересылать новые посты в Discord.

## 🛡️ Безопасность

- **НИКОГДА не публикуйте токены** (Discord, Telegram) в публичных местах или в истории git.
- Файл `.env` добавлен в `.gitignore`, чтобы случайно не закоммитить секреты.
